== Description ==

Pulls information from a list of Stack Overflow users and displays their reputation and badges and sorts them by descending order.

== Installation ==

1. Upload `StackOverflowLeaderboard` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the 'Widgets' page and drag it to the appropriate section
4. Manage the user list from the widget
